# Tesina de grado
Se desarrollo un modelo de IA basado en Redes Convolucionales en 3D con la arquitectura R(2+1)D-18 preentrenada en Kinetics-400 para la deteccion de tecnicas bien realizadas y posiblemente riesgosas en el ejercicio de  peso muerto convencional. Luego de esto se desarrollo una app de escritorio minimalista para ver el modelo funcionando
